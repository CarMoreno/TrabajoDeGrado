Cumpliendo con los estándares de la W3C para la publicación de datasets en la Web de Datos se dispone de un SPARQL Enpoint, una herramienta que permite realizar consultas SPARQL sobre un grafo de entrada compuesto por tripletas RDF. En este caso, el grafo de entrada es el dataset ecoturístico usado en toda la aplicación. Acceder al SPARQL Endpoint

A continuación se muestran algunas consultas que pueden realizarse en el SPARQL Endpoint

Consulta 1

# Selecciona una direccion de las empresas # de la Ruta del Maíz. PREFIX UMBEL: <http://umbel.org/umbel#> PREFIX VCARD: <http://www.w3.org/2006/vcard/ns#> PREFIX MZ: <http://190.14.254.237/dataseteco/RutaDelMaiz/> SELECT ?direccion WHERE { ?sub UMBEL:isRelatedTo MZ:Empresas. ?sub VCARD:adr ?cualquiera . ?cualquiera VCARD:street-address ?direccion . }

Consulta 2

# Selecciona la hora en que abre y cierra los restaurantes de la Ruta del Maíz PREFIX UMBEL: <http://umbel.org/umbel#> PREFIX FOAF: <http://xmlns.com/foaf/0.1/> PREFIX GR: <http://purl.org/goodrelations/v1#> PREFIX MZ: <http://190.14.254.237/dataseteco/RutaDelMaiz/> SELECT ?abre ?cierra WHERE { ?sub UMBEL:isRelatedTo MZ:Restaurantes. ?sub FOAF:depiction ?cualquier . ?sub GR:hasOpeningHoursSpecification ?obj2 . ?obj2 GR:opens ?abre . ?obj2 GR:closes ?cierra . }

Consulta 3

# Selecciona el numero de habitaciones, de camas y la # direccion de todos los hoteles de la Ruta del Anillo # Agricola. PREFIX UMBEL: <http://umbel.org/umbel#> PREFIX ACCO: <http://purl.org/acco/ns#> PREFIX FOAF: <http://xmlns.com/foaf/0.1/> PREFIX VCARD: <http://www.w3.org/2006/vcard/ns#> PREFIX GR: <http://purl.org/goodrelations/v1#> PREFIX JB: <http://190.14.254.237/dataseteco/RutaDelAnilloAgricola/> SELECT ?direccion ?numHabitaciones ?numCamas WHERE { ?sub UMBEL:isRelatedTo JB:Alojamientos. ?sub FOAF:depiction ?cualquier . ?sub VCARD:adr ?obj1 . ?obj1 VCARD:street-address ?direccion . ?obj2 ACCO:partOf ?sub . ?obj2 ACCO:numberOfRooms ?uriValue . ?uriValue GR:hasValue ?numHabitaciones . ?obj2 ACCO:bed ?uriBed . ?uriBed ACCO:quantity ?numCamas . }

Consulta 4

# Selecciona las direcciones de Alojamientos, Empresas y Restaurantes # de la Ruta del Maíz PREFIX UMBEL: <http://umbel.org/umbel#> PREFIX VCARD: <http://www.w3.org/2006/vcard/ns#> PREFIX MZ: <http://190.14.254.237/dataseteco/RutaDelMaiz/> SELECT DISTINCT ?direccion WHERE { { ?sub UMBEL:isRelatedTo MZ:Empresas. ?sub VCARD:adr ?obj1 . } UNION { ?sub UMBEL:isRelatedTo MZ:Alojamientos. ?sub VCARD:adr ?obj1 . } UNION { ?sub UMBEL:isRelatedTo MZ:Restaurantes. ?sub VCARD:adr ?obj1 . } ?obj1 VCARD:street-address ?direccion . }
