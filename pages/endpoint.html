Cumpliendo con los estándares de la W3C para la publicación de datasets en la Web de Datos se dispone de un SPARQL Enpoint, una herramienta que permite realizar consultas SPARQL sobre un grafo de entrada compuesto por tripletas RDF. En este caso, el grafo de entrada es el dataset ecoturístico usado en toda la aplicación. A continuación se muestran algunas consultas que pueden realizarse en el SPARQL Endpoint

Consulta 1

# Selecciona una direccion de las empresas # de la Ruta del Maíz. PREFIX UMBEL: <http://umbel.org/umbel#> PREFIX VCARD: <http://www.w3.org/2006/vcard/ns#> PREFIX MZ: <http://190.14.254.237/dataseteco/RutaDelMaiz/> SELECT ?direccion WHERE { ?sub UMBEL:isRelatedTo MZ:Empresas. ?sub VCARD:adr ?cualquiera . ?cualquiera VCARD:street-address ?direccion . }
